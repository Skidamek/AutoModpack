### Fichier de Configuration Client

`/automodpack/automodpack-client.json`

| Nom                  | Valeur par Défaut | Description                                                                                                                                                        |
|---------------------|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `DO_NOT_CHANGE_IT`  | `<file version>`  | Numéro de version du fichier utilisé pour la conversion automatique des anciennes versions de configuration vers les nouvelles lors de la mise à jour d'AutoModpack.  |
| `selectedModpack`   |                   | Le dossier principal du modpack auquel vous voulez jouer. Taper le nom du modpack ici provoquera son chargement.                                            |
| `installedModpacks` |                   | Une liste des modpacks qui sont installés sur le client.                                                                                                       |
| `selfUpdater`       | `false`           | Active/désactive les mises à jour automatiques d'AutoModpack. Cela n'affecte pas l'activité du mod dans l'installation des modpacks. Cela ne désactive pas la synchronisation de la version d'AutoModpack avec le serveur. |
